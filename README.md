Overview

This project presents a complete pipeline for the 3D reconstruction of an agricultural environment using a mobile robot equipped with an RGB-D camera.

The main objective is to generate a digital representation of an agricultural field that can support precision agriculture applications such as:

Crop monitoring;
Biomass estimation;
Vegetation structure analysis;
Agricultural planning.

The project was developed as part of a university robotics course and combines robotics, computer vision and 3D reconstruction techniques.
